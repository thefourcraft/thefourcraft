What is Web Design?

Web design refers to the design of websites that are displayed on the internet.
Web design is what creates the overall look and feel when you’re using a website.

It’s the process of planning and building the elements of your website, from structure and layout to images,
colors, fonts, and graphics.

Web design has numerous components that work together to create the finished experience of a website,
including graphic design, user experience design, interface design, search engine optimization (SEO),
and content creation.
These elements determine how a website looks, feels, and works on various devices

I personally think that security of the website code and speed/optimization is also part of web design
