The Minecraft Hebrew Localization 

Minecraft is a sandbox video game developed by Mojang Studios.
The game was created by Markus "Notch" Persson in the Java programming language.
Following several early private testing versions, it was first made public in May 2009 before being fully released in November 2011, with Notch stepping down and Jens "Jeb" Bergensten taking over development.

During the game's first years (2009-2015) the game was getting popular and people started to translate the game into different languages... the first one was Russian and Spanish but then slowly more languages were added int the game

I and a small group of people committed the first translation back in 2012 using google translate because we didn't know English, to say that the translations were bad is an understatement.

After that, the project was on and off and by 2017, Mojang took the localization of the game bit more seriously and started the initial Minecraft localization project.

This allowed us better collaboration and improvement of the Hebrew localization within the game.
